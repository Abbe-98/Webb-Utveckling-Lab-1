# MathRush – Math Game

## Description:
A fast-paced math game where the player must answer as many math questions as possible within 3 minutes.
Responsiveness Requirement
The layout must:
Adjust to different screen sizes (mobile, tablet, desktop)
Use responsive CSS units (%, em, rem, vw/vh)
Be built using Flexbox and/or CSS Grid
Support both portrait and landscape modes (especially for mobile)

### Git Repository Structure
Your GitHub repository should include:
main branch (production-ready)
At least two feature branches, e.g.:
feature/game-logic
feature/ui-layout

### Goals (Step-by-step breakdown)
1. Start Page
A single button:
"Start Game"
2. Player Setup Screen
A text field for entering the player's name
OBS: The player name will be the name that they are compiting with the entire session;
Placeholder: "Enter your name"
Radio buttons to select difficulty level:
Easy
Hard
A button:
"Save and Continue"
4. Game Launch Screen
Display the player’s name and selected difficulty level
A button to actually begin the 3-minute game:
"Start Game for Real"
When game is over the player has either won or lost
if win the player will go to the high score list and at the bottom there will be tow buttons either start again or back to menu
if lose the player will go to asside where they can choose to play again or back to menu with a sad pic.
if player chose back to menu restart this process.

### 🛠️ Technologies
- HTML5
- CSS3 (flexbox, grid, modern animations)
- JavaScript (ES6)
